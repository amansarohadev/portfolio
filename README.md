## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on
your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your
machine.

### Installation

1.  **Clone the repository:**

``` bash
git clone https://github.com/amansarohadev/portfolio.git
```

2.  **Navigate to the directory:**

``` bash
cd portfolio
```

3.  **Install dependencies:**

``` bash
npm install
```

4.  **Start the development server:**

``` bash
npm run dev
```

> *Your site will now be running at `http://localhost:5173` (or the port
> Vite provides).*
